LocalTunnel Server

Expose your localhost to the world with ease

Quick Start

1. Install the Client

npm install -g localtunnel

2. Create a Tunnel

lt --port 3000 --host https://your-server-domain

3. Using the API

Create a Tunnel

curl -X POST https://your-server-domain/api/tunnels

Get Tunnel Info

curl https://your-server-domain/api/tunnels/{tunnel_id}

Delete a Tunnel

curl -X DELETE https://your-server-domain/api/tunnels/{tunnel_id}

4. Node.js Example

const localtunnel = require('localtunnel');

(async () => {
  const tunnel = await localtunnel({ 
    port: 3000,
    host: 'https://your-server-domain'
  });

  console.log('Tunnel URL:', tunnel.url);
})();
